## 🌙 فكرة المشروع

**سَنَد** هو تطبيق إسلامي شامل يهدف إلى مرافقة المسلم في رحلته اليومية الروحانية، من خلال توفير أدوات متكاملة تجمع بين الأصالة الإسلامية والتصميم العصري الأنيق.

الهدف من التطبيق هو أن يكون **سنداً** حقيقياً للمسلم في يومه، يذكّره بمواقيت الصلاة، يُعينه على تلاوة القرآن الكريم، ويُيسّر عليه قراءة الأذكار وتسبيح الله — كل ذلك في تجربة مستخدم سلسة وجذابة بصرياً مستوحاة من التراث الإسلامي الجميل.

---

## ✨ المزايا والميزات

### 🏠 الشاشة الرئيسية
- **تحية شخصية** مع عرض التاريخ الهجري الحالي
- **اقتبس اليوم** — حديث نبوي شريف يتجدد يومياً
- **كاروسيل ديناميكي** (5 صور) يعرض وقت الصلاة الحالي مع شريط تقدم ذهبي ومتأثرات زجاجية (Glassmorphism)
- **قائمة مواقيت الصلاة** — عرض أفقي قابل للتمرير لأوقات اليوم والغد
- **وصول سريع** لأبرز الميزات (القرآن، المسبحة، الأذكار)
- **رحلتي اليومية** — تتبع تقدم المهام اليومية بأسلوب تحفيزي (Gamification) مع شارات وإحصائيات

### 📖 القرآن الكريم
- تصفح وقراءة آيات القرآن الكريم
- واجهة تلاوة نظيفة وسهلة القراءة

### 🕌 مواقيت الصلاة
- عرض أوقات الصلوات الخمس
- **جدول شهري** لمواقيت الصلاة بعرض منظّم

### 📿 المسبحة الرقمية (التسبيح)
- عداد تسبيح رقمي تفاعلي
- تتبع عدد التسبيحات بسهولة

### 🤲 الأذكار
- أذكار الصباح والمساء
- أذكار متنوعة منظمة بشكل واضح

### 👤 الملف الشخصي
- إدارة بيانات المستخدم وإعدادات التطبيق
- دعم الوضع المظلم (Dark Mode)

## 🏗️ البنية المعمارية

يتبع المشروع نمط **Clean Architecture - Feature First**:

```
lib/
├── core/                          # المكونات المشتركة
│   ├── theme/                     # نظام الألوان والثيمات
│   │   ├── app_colors.dart        # لوحة الألوان الأساسية
│   │   └── app_theme.dart         # تعريف الثيمات (فاتح/مظلم)
│   └── common_widgets/            # الودجات المشتركة
│       ├── prayer_time_item.dart  # عنصر وقت الصلاة
│       ├── quick_action_card.dart # بطاقة الوصول السريع
│       └── islamic_icons.dart     # أيقونات إسلامية مخصصة
│
├── features/                      # الميزات (Feature-First)
│   ├── home/                      # الشاشة الرئيسية
│   ├── quran/                     # القرآن الكريم
│   ├── prayer_times/              # مواقيت الصلاة
│   ├── azkar/                     # الأذكار
│   ├── tasbeeh/                   # المسبحة
│   ├── profile/                   # الملف الشخصي
│   ├── explore/                   # الاستكشاف
│   ├── groups/                    # المجموعات
│   ├── activities/                # الأنشطة
│   └── main/                      # الهيكل الرئيسي (Scaffold + Navigation)
│
├── routes.dart                    # إدارة التنقل (GoRouter)
└── main.dart                      # نقطة الدخول
```

## 🚀 تشغيل المشروع
